# Ecommerce-api

## Introduction
This is an API for an e-commerce platform that allows users to manage products, orders, and customers.

## Installation
To install the dependencies, run:
```bash
npm install
```

## Usage
To start the server, run:
```bash
npm start
```
The server will start on `http://localhost:3000`.
